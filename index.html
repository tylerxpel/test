Your Computer

Outlook Version

New Outlook & Outlook on the Web — Windows

outlook.com · Microsoft 365 · New Outlook for Windows

  1. 1Click the Settings gear icon in the top-right corner of Outlook.
  2. 2Select Accounts → Signatures.
  3. 3Click New signature (or the + icon) and give it a name (e.g. XPEL).
  4. 4Click into the signature editor, then press Ctrl+V to paste.
  5. 5Click Save. Set it as your default for new messages and/or replies using the dropdowns.

Classic Outlook Desktop — Windows

Outlook 2016 · 2019 · 2021 (Windows)

  1. 1Open Outlook and go to File → Options → Mail.
  2. 2Click Signatures… under the Compose messages section.
  3. 3Click New, enter a name (e.g. XPEL), and click OK.
  4. 4Click into the editor area at the bottom, then press Ctrl+V to paste.
  5. 5Click OK to save. Use the dropdowns at top right to set it as default for New messages and/or Replies/forwards.

New Outlook & Outlook on the Web — Mac

outlook.com · Microsoft 365 · New Outlook for Mac

  1. 1Click the Settings gear icon in the top-right corner. If you don’t see one, click the Outlook menu → Settings instead.
  2. 2Select Accounts → Signatures.
  3. 3Click New signature (or the + icon) and give it a name (e.g. XPEL).
  4. 4Click into the signature editor, then press ⌘V to paste.
  5. 5A small clipboard icon will appear right after pasting — click it and choose Keep Source Formatting. Skipping this is the most common cause of a squished or distorted signature on Mac.
  6. 6Click Save. Set it as your default for new messages and/or replies using the dropdowns.

Classic Outlook for Mac

Outlook 2019 · 2021 for Mac (pre-“New Outlook” interface)

  1. 1Click the Outlook menu in the top-left of the screen, then select Preferences.
  2. 2Click Signatures under the Email section.
  3. 3Click the + icon to add a new signature and give it a name (e.g. XPEL).
  4. 4Click into the editor, then press ⌘V to paste.
  5. 5A small clipboard icon will appear right after pasting — click it and choose Keep Source Formatting. Skipping this is the most common cause of a squished or distorted signature on Mac.
  6. 6Click Save, close the window, then use the dropdowns to set it as default for New messages and/or Replies/forwards.

Mobile Setup

Outlook Mobile signatures are separate from desktop and don’t sync automatically — you'll need to set this up once on each phone. This is a simplified, plain-text signature that works the same way on iOS and Android, and it's clearly marked as a mobile reply.

Plain text, no logo or images — pastes cleanly into Outlook Mobile on any phone.

Outlook Mobile — iOS & Android

Same steps on both platforms

  1. 1Tap the account icon or menu in the top-left of Outlook, then tap the Settings gear.
  2. 2Select your account, then tap Signature.
  3. 3Clear any existing text, tap and hold the empty field, then choose Paste.
  4. 4Tap the checkmark (or Done) in the top-right to save.
